# ⚽ Network Science: Analyzing El Clásico (2009/2010)

This folder contains a sports analytics project developed for the **Networks** course. Inspired by the research paper *"Defining a historic football team: Using Network Science to Analyze Guardiola’s FC Barcelona,"* this project models player interactions as passing networks.

---

### 📊 Project Overview
The study analyzes the strategic passing patterns of **FC Barcelona vs. Real Madrid** during the 2009/2010 La Liga season. By treating players as **nodes** and passes as **weighted edges**, we can visualize and quantify the "Tiki-Taka" philosophy versus Madrid's direct play.
